# 🎬 Hackpad For Editing

A 6-key mechanical macropad built for video/audio editors.
Plug-and-play USB HID — no drivers needed.



![Hackpad](assets/hackpad.jpg)



## What It Does

- 6 mechanical switches mapped to editing shortcuts
- Works as standard USB keyboard (Pro Micro + Keyboard.h)
- INPUT_PULLUP — no external resistors needed

## Key Layout
hackpad-for-editing/
├── Firmware/   → main.cpp
├── CAD/        → STL files
├── PCB/        → KiCad files
└── assets/     → Photos



## How to Flash

1. Install PlatformIO in VS Code
2. Open Firmware/ folder
3. Connect Pro Micro via USB
4. Click Upload ▶️
