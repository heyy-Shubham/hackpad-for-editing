🚀 Hackpad For Editing

This project is developed using PlatformIO IDE extension in Visual Studio Code.
It is designed for efficient code editing and embedded development workflows.

🛠️ Tech Stack

- PlatformIO
- C/C++
- Visual Studio Code

📌 Features

- Fast and lightweight editing
- Organized project structure
- Easy build and upload support using PlatformIO

▶️ How to Use

1. Open the project in VS Code
2. Install PlatformIO extension
3. Build and upload using PlatformIO tools
